使用方法

http://bbs.wiz.cn/thread-6860-1-1.html

特性
--------
设置Wiz文章对应的wiki文件名、本地版本库路径、服务器库路径、归档用的脚本。
自动保存.wiki文件，并更新到Google Code服务器。（如果用hg，即使不联网，也能更新到本地库，避免历史信息丢失）
预览Wiz中的符合wiki语法的文章。
按习惯的所见即所得的方式插入图片到Wiz文章，发布时图片自动发布到服务器，并能正确显示到Wiki页面上。


注意事项
---------
1、编写文章要以Google Code语法的格式。
2、本地编辑时不是所见即所得，预览效果不是非常好。
3、每篇文章要设置本地路径、远程路径，稍微有点麻烦。
